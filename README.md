<h1>School Fund Mangement</h1>
School Fund Management System
A modern and responsive web-based School Fund Management System built using HTML, Tailwind CSS, and JavaScript. This project helps schools manage student records, teacher payroll, school funds, financial transactions, and reports through a clean and user-friendly dashboard interface.
Features


📊 Interactive Dashboard


Overview of total students


Teacher count


Available funds


Salary budget




🎓 Student Records Management


Add and manage student details


Track fee payment status


Organized student data tables




👨‍🏫 Teacher Payroll Management


Add teacher records


Manage subject and salary details


Payroll tracking system




💰 School Fund Management


Add income and expense transactions


Financial transaction history


Budget monitoring




📑 Reports Section


Generate downloadable reports


Fee reports


Salary reports


Full financial summaries




🎨 Modern UI Design


Fully responsive layout


Built with Tailwind CSS


Professional dashboard interface




Technologies Used


HTML5


Tailwind CSS


JavaScript


Project Goals
This project was developed to simplify school financial management by providing an easy-to-use digital platform for maintaining records, tracking funds, and managing payroll efficiently.
